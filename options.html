JavaScript

Все скрипты автоматически запускаются с событием DOMContentLoaded.

Опция программное подключение запускает скрипты через встроенную в браузер функцию executeScript. Такие скрипты имеют полный доступ к DOM дереву, но контекст страницы им недоступен. Такой метод запуска позволяет избежать проблем, например, с CORS.

CSS стили

CSS включаются практически мгновенно, еще перед полным построением DOM дерева.

Опция максимальный приоритет автоматически добавляет !important ко всем свойствам перед применением на страницу.

Опция программное подключение использует стандартный для браузера способ внедрения стилей insertCSS. Стили запущеные таким образом имеют высший приоритет перед стилями страницы (при использовании !important – максимальный), однако "живые" правки через инструменты разработчика исключены.

URL

Examples of valid masks:

  • example.com
  • example.com, seconddomain.com
  • example.com, !example.com/excluded
  • example.com/*/images/
  • http://dev.example.com/ — recommended

JavaScript

All scripts automatically start with the event DOMContentLoaded.

The programmatic injection option runs scripts through the built-in function executeScript. These scripts have full access to the DOM tree, but not to the context of the page. This method avoids CORS issues.

CSS styles

CSS is included almost instantly, even before the DOM ready.

The highest priority option automatically adds !important to all CSS properties before applying to the page.

The programmatic injection option use the standard way for the browser to inject styles insertCSS. This way injecting styles have the high priority over page styles (with !important – highest), however, "live" edits through developer tools are excluded.
